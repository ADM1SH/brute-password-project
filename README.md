# Brute Force Password Project

## Overview
A C-based implementation of a brute-force password cracker. This project demonstrates basic concepts of string manipulation, loops, and trial-and-error algorithm design in C.

## Files
- `password.c`: Main source code containing the logic for the brute-force attack.

## How to Run
Compile with a C compiler:
```bash
gcc password.c -o password
./password
```
